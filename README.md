# Insurance Acceptance Optimization (Seguro Auto)

## 📌 Problem
This project aims to help an insurance company decide which customers to accept in order to maximize financial performance.

## 🧠 Approach
- Predict probability of total loss using Machine Learning
- Model used:
  - Random Forest
- Convert predictions into business decisions

## 💰 Key Idea

Instead of using a fixed threshold (e.g. 0.5), decisions are based on expected profit:

lucro = premio_seguro - probabilidade * valor_segurado

## 🚀 Outcome
The model enables data-driven decision making focused on profitability rather than accuracy.
